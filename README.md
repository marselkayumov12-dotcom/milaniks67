## 📖 Description

Random Task Generator is a desktop GUI application that helps you overcome decision paralysis by randomly suggesting tasks from different categories. Perfect for:

- 🏃 Daily routine planning
- 📚 Study session variety
- 💼 Work task randomization
- 🎨 Adding spontaneity to your day

---

## ✨ Features

- ✅ **Random task generation** from predefined database
- ✅ **4 categories**: Study 📚, Sport 🏃, Work 💼, Other 🎨
- ✅ **Add custom tasks** to the database
- ✅ **Filter history** by category
- ✅ **Persistent history** saved to JSON file
- ✅ **Auto-save** every generated task
- ✅ **Input validation** (no empty tasks)
- ✅ **Statistics** display (total tasks generated)
